Simple Weather App

This is the simple weather app made using HTML, CSS and JS.
This weather app shows the temperature in celsius, humidity level and the speed of wind. 
Also the weather icon is changing depending on the current weather conditions and if it's a day or a night.

To run this weather app properly, download the folder, place it in your computer and make sure that the 
all the patterns to the folder in the code are indicated right (use Visual Studio Code to open the code), then 
run the index file and weather app shoud work.
